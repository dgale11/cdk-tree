Tree

Tree is a Angular Material component that enables to build a customized tree experience for structured data.

There are two types of Trees in Angular Material, one of those is Flat and the other one is Nested.

Flat Tree

The logical parent-child relationship is flattened and nodes are not rendered inside of each other, but instead are rendered as siblings in sequence. Can handle lots of data.

Nested Tree

In nested tree, children nodes are placed inside their parent node in the DOM. The parent node contains a node outlet into which children are projected.
